Bem-vindo ao sistema de agenda da Congregação Batista de Árvores Verdes no PSH!


Este sistema foi desenvolvido para facilitar o gerenciamento da escala de cultos e aniversariantes do mês.


Como usar:


  1. Para adicionar ou editar informações de um dia específico, clique na célula correspondente no calendário.
  2. Preencha os campos necessários, como Dirigente, Pregador, Evento, Atalaias e Ministros Infantis.
  3. Clique em "Registrar Dia" para salvar as informações.
  4. Use os botões na barra lateral para gerar um PDF da agenda ou apagar registros conforme necessário.

  • As informações de aniversariantes do mês serão exibidas automaticamente.
  • Para salvar como PDF, ao aparecer a tela de impressão, selecione a opção "Salvar como PDF" no seu navegador.
  • Apagar registros antigos é fácil! Use o botão "Apagar Tudo" para remover todas as informações de uma vez.
  • Se houver divergências, tente usar o navegador Edge da Microsoft.

    • Explore as funcionalidades e aproveite a praticidade do sistema!
